# git-agent

Generic git skills for non-engineers. Clone any repo, contribute changes — no git knowledge needed.

## Skills

- `/setup` — one-time machine setup (git install, GitHub SSH auth)
- `/clone` — clone a repo you want to work on
- `/contribute` — commit and push your changes

## Typical flow

```
/setup       ← once per machine
/clone       ← once per repo
/contribute  ← whenever you want to share changes
```

## Scripts

- `scripts/contribute.sh` — Mac/Linux
- `scripts/contribute.ps1` — Windows
